Online Payment Fraud Detection

Une application pour la prédiction de fraude basée sur l'apprentissage automatique.

Commencer

Logistic Regression

La régression logistique est un modèle statistique utilisé pour prédire la probabilité d'un événement en fonction de un ou plusieurs facteurs.

Random Forest

Random Forest est un algorithme d'apprentissage automatique utilisé à la fois pour la classification et la régression. Il crée un ensemble de modèles d'arbres de décision, puis agrège leurs prédictions.

Oversampling

L'oversampling est une technique utilisée pour équilibrer les classes dans un ensemble de données déséquilibré en augmentant le nombre d'échantillons de la classe minoritaire.
